# LURI
此数据集由无人机低空拍摄路面获得，包含16条公路的图像，按照字母表顺序分别存放于16个文件夹中，每个文件夹中有图和json两个文件夹，分别存放原图和对应的json图像，像素统一为640×480。

This dataset was obtained by shooting road surfaces at low altitudes with drones. It contains images of 16 highways, which are stored in 16 folders in alphabetical order. Each folder has two folders: image and json, which store the original image and the corresponding json image respectively. The pixels are uniformly 640×480.
